Exercise 1

Analyse the logical forms of the following statements:

a

We'll have either a reading assignment or homework problems, but we won't have both homework problems, but we won't have both homework problems and a test.

Solution

If we define the statement "we'll have a reading assignment" as \(R\) and "we'll have homework problems" as \(H\) then we can write the statement as \[(R \lor H) \land \neg (R \land H)\]

b

You won't go skiing, or you will and there won't be any snow.

Solution

If we define "you will go skiing" as \(K\) and "there will be snow" as \(S\) then we can write the statement as \[\neg K \lor (K \land \neg S)\]

c

\[\sqrt{7} \nleq 2\]

Solution

If we define \(\sqrt{7} \lt 2\) as \(A\) and \(\sqrt{7} = 2\) as B then we can write the statement as \[\neg(A \land B)\]

Exercise 2

Analyze the logical forms of the following statements:

a

Either John and Bill are both telling the truth, or neither of them is.

Solution

If we define "John tells the truth" as \(J\) and "Bill tells the truth" as \(B\) then we can write the statement as \[(J \land B) \lor (\neg J \land \neg B)\]

b

I'll have either fish or chicken, but I won't have both fish and mashed potatoes.

Solution

If we define "I'll have fish" as \(F\) and "I'll have chicken" as \(C\) and "I'll have mashed potatoes" as \(M\) then we can write the statemement as \[(F \lor C) \land \neg (F \land M)\]

c

3 is a common divisor of 6, 9 and 15

Solution

If we define "\(3\) divides \(6\)" as \(A\) and "\(3\) divides \(9\)" as \(B\) and "\(3\) divides \(15\)" as \(C\) then we can write the statement as \[A \land B \land C\]
