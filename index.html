Exercise 1

Analyse the logical forms of the following statements:

a

We'll have either a reading assignment or homework problems, but we won't have both homework problems and a test.

Solution

If we let \(R\) stand for the statement "we'll have a reading assignment" as and let \(H\) stand for the statement "we'll have homework problems" then we can write the statement as \[(R \lor H) \land \neg (R \land H)\]

b

You won't go skiing, or you will and there won't be any snow.

Solution

If we let \(K\) stand for the statement "you will go skiing" and let \(S\) stand for the statement "there will be snow" then we can write the statement as \[\neg K \lor (K \land \neg S)\]

c

\[\sqrt{7} \nleq 2\]

Solution

If we let \(A\) be \(\sqrt{7} \lt 2\) and let B be \(\sqrt{7} = 2\) then we can write the statement as \[\neg(A \land B)\]

Exercise 2

Analyze the logical forms of the following statements:

a

Either John and Bill are both telling the truth, or neither of them is.

Solution

If we let \(J\) stand for the statement "John tells the truth" and let \(B\) stand for the statement "Bill tells the truth" then we can write the statement as \[(J \land B) \lor (\neg J \land \neg B)\]

b

I'll have either fish or chicken, but I won't have both fish and mashed potatoes.

Solution

If we let \(F\) stand for the statement "I'll have fish" and let \(C\) stand for the statement "I'll have chicken" and let \(M\) stand for the statement "I'll have mashed potatoes" then we can write the statement as \[(F \lor C) \land \neg (F \land M)\]

c

3 is a common divisor of 6, 9 and 15

Solution

If we let \(A\) be "\(3\) divides \(6\)" and let \(B\) be "\(3\) divides \(9\) and let \(C\) be "\(3\) divides \(15\)" then we can write the statement as \[A \land B \land C\]

Exercise 3

Analyze the logical forms of the following statements:

a

Alice and Bob are not both in the room.

Solution

If we let \(A\) stand for "Alice is in the room" and let \(B\) stand for "Bob is in the room" then we can write the statement as \[\neg (A \land B)\]

b

Alice and Bob are both not in the room.

Solution

If we let \(A\) stand for "Alice is in the room" and let \(B\) stand for "Bob is in the room" then we can write the statement as \[(\neg A \land \neg B)\]

c

Either Alice or Bob is not in the room

Solution

If we let \(A\) stand for "Alice is in the room" and let \(B\) stand for "Bob is in the room" then we can write the statement as \[\neg A \lor \neg B\]

d

Neither Alice or Bob is in the room

Solution

If we let \(A\) stand for "Alice is in the room" and let \(B\) stand for "Bob is in the room" then we can write the statement as \[(\neg A \land \neg B)\]

Exercise 4

Analyze the logical forms of the following statements:

a

Either both Ralph and Ed are tall, or both of them are handsome.

Solution

If we let \(R\) stand for "Ralph is tall", \(E\) for "Ed is tall", \(S\) for "Ralph is handsome" and \(F\) for "Ed is handsome" then we can write the statement as \[(R \land E) \lor (S \land F)\]

b

Both Ralph and Ed are either tall or handsome.

Solution

If we let \(R\) stand for "Ralph is tall", \(E\) for "Ed is tall", \(S\) for "Ralph is handsome" and \(F\) for "Ed is handsome" then we can write the statement as \[(R \lor S) \land (E \lor F) \land \neg (R \land S) \land \neg (E \land F)\]

c

Both Ralph and Ed are neither tall nor handsome

Solution

If we let \(R\) stand for "Ralph is tall", \(E\) for "Ed is tall", \(S\) for "Ralph is handsome" and \(F\) for "Ed is handsome" then we can write the statement as \[\neg R \land \neg E \land \neg S \land \neg F\]

d

Neither Ralph nor Ed is both tall and handsome

Solution

If we let \(R\) stand for "Ralph is tall", \(E\) for "Ed is tall", \(S\) for "Ralph is handsome" and \(F\) for "Ed is handsome" then we can write the statement as \[\neg (R \land S) \land \neg (E \land F)\]

Exercise 5

Which of the following expressions are well-formed formulas?

a

\[\neg (\neg P \lor \neg \neg R)\]

Solution

Well-formed

b

\[\neg (P, Q, \land R)\]

Solution

Not well-formed. Comma's are not part of the language of sentential logic.

c

\[P \land \neg P\]

Solution

Well-formed

d

\[(P \land Q)(P \lor R)\]

Solution

Not well-formed. There should be a logical connective in between \()(\).

Exercise 6

Let \(P\) stand for the statement "I will buy the pants" and \(S\) for the statement "I will buy the shirt". What English sentences are represented by the following formulas?

a

\[\neg (P \land \neg S)\]

Solution

I won't buy the pants and not buy the shirt.

b

\[\neg P \land \neg S\]

Solution

I won't buy the pants and I won't buy the shirt.

c

\[\neg P \lor \neg S\]

Solution

I won't buy the pants or I won't buy the shirt.
